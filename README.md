# ICI Tools

Various tools for working with ICI files 

## Converter

Converts image files (PNGs, JPGs, etc) to ICIs

### Usage

```
ici_converter [OPTIONS] <FILE>

Arguments:
  <FILE>  Image file to convert

Options:
  -o, --output <FILE>  
  -h, --help           Print help
  -V, --version        Print version
```

## Viewer 

Displays ICI files

### Usage

```
ici_viewer <FILE>

Arguments:
  <FILE>  Image to show

Options:
  -h, --help     Print help
  -V, --version  Print version
```
